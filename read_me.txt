Trivia App
- By Yash J.


------------------------------------------------------------------------------------------------------------------------------------------

App name = Trivia
Architecture followed = MVP
Language used = Kotlin


------------------------------------------------------------------------------------------------------------------------------------------

Simple app to take user inputs through text fields, radio buttons, checkboxes, etc. And show a summary of user input data.
App also features a list of all history trivia games that user have played and saves it in the local environment.


------------------------------------------------------------------------------------------------------------------------------------------

App flow:

Splash
|
User name screen
|
Cricketer and Flag question screens in sequence
|
Summary Screen---------------------------------------------------
|			|					|
Finish			History					Back click
(Restart Game) 		(Previously played games)		(Exit App)



------------------------------------------------------------------------------------------------------------------------------------------

Important Directories:

root/apk  ---- pre generated APK file to directly install and check the application
root/screens ---- in-app screenshots of all the screens
